Personal Project: Jot - A Scheduling App

  • React
  • REST API
  • NodeJS
  • Express
  • MongoDB

About

A full stack event scheduling application that lets users create and share event details with their guests who can RSVP to the event. It also allows editing RSVP information and dynamically updates the RSVP information upon a change.

What is so cool about it?

Real people are using this app to schedule real events. Granted they are my friends and we use it to schedule our weekly ball game but still the app is really neat and it has been thoroughly tested. I am even getting requests to generate an attendance report.

Personal Project: FoodGenie

  • React
  • FoodData Central API
  • TypeScript

About

A food search engine made with React which returns the ingredients and nutrition info of food items powered by FoodData Central API from U.S. Department of Agriculture.

What is so cool about it?

The FoodData Central API from the USDA is awesome. It is blazing fast and has information about more than 400,000 food items. Go check it out.
